myFlix Documentation

This API was designed to cultivate the server-side component of a movie web application. Users will be able to create a personal profile where they can create a list of their favorite movies as well as access information on movies such as directors and genres.

Operation Description Endpoint URL Method Type Query Parameters Format Of Data In Body Of Request Format Of Data In Body Of Response
Return a list of ALL movies to the user /movies GET None None Returns an array of all movies
Return data (description, genre, director, image URL, whether it/'s featured or not) about a single movie by title to the user movies/:title GET :title None Returns a JSON object with data about a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”) movies/genres/:name GET :name None Returns a JSON object with data about a single genre
Return data about a director (bio, birth year, death year) by name movies/directors/:name GET :name None Returns a JSON object with data about a single director
Allow new users to register /users POST None { username: 'JohnnyDoe', password: 'HeresJohnny' email: 'Boutthebread', birthday: "1985-08-08" } { username: 'JohnnyDoe', password: 'HeresJohnny', email: 'Boutthebread', birthday: "1985-08-08", id: '3jk3ljfk' }
Allow users to update their user info (username) /users/:username/:password PUT :username, :password { password: 'HeresJohnny' } { username: 'JohnnyDoe', password: 'HeresJohnny', id: '3jk3ljfk' }
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:username/:title POST :username, :title None '${movieTitle} has been added to user ${username}s array'
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:username/:title DELETE :username, :title None '${movieTitle} has been removed from user ${username}s array'
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:username DELETE :username None '${username} has been removed as a user'
